## Version 1.0 - Initial Release

This release introduces a comprehensive 90-minute hands-on lab for building Retrieval-Augmented Generation (RAG) pipelines from scratch. The module uses a zero-install approach with Google Colab to eliminate infrastructure complexity and focus on RAG implementation logic.

### Release details

#### Additions

- **Zero-Install Environment:** Complete RAG development using only browser-based tools
- **Two-Part Structure:** 45-minute microlessons covering setup and complete implementation
- **Real-World Dataset:** Support ticket data for practical RAG scenarios
- **Step-by-Step Code:** Modular code snippets for each RAG pipeline component
- **Hands-On Challenges:** Interactive exercises for parameter tuning and optimization
- **Production Considerations:** Evaluation metrics, cost analysis, and scaling strategies
- **Comprehensive Instructor Guide:** Detailed teaching notes and troubleshooting guidance

#### Technical Implementation

- **LangChain Framework:** Modern RAG orchestration with LCEL syntax
- **FAISS Vector Store:** In-memory vector database for semantic search
- **OpenAI Integration:** Embeddings (text-embedding-ada-002) and chat models (GPT-3.5-Turbo)
- **Interactive Experimentation:** Multiple challenge scenarios for practical learning
- **Evaluation Framework:** Basic precision metrics and similarity score analysis

#### Learning Outcomes

- Understanding of five core RAG steps: Load → Chunk → Embed → Retrieve → Generate
- Practical experience with vector similarity search and semantic retrieval
- Hands-on implementation of complete end-to-end RAG chains
- Optimization techniques for chunk size, retrieval parameters, and prompt engineering
- Production readiness considerations including cost management and scalability
